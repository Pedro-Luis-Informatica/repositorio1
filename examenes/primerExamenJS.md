# 🚀 Lista de Ejercicios de JavaScript Inicial

---

1. Crea un programa que pida el nombre al usuario mediante un `prompt` y lo salude con un `alert`.
2. Solicita un número al usuario y muestra un mensaje indicando si es un número positivo o negativo usando `if`.
3. Pregunta al usuario su edad; si es mayor o igual a 18, muestra un mensaje de "Acceso permitido".
4. Pide un número y determina si es par o impar utilizando el operador de residuo (%).
5. Utiliza el **operador ternario** para verificar si un número ingresado es mayor a 100.
6. Solicita dos números distintos y muestra mediante un `alert` cuál de los dos es el mayor.
7. Crea un sistema de validación que pida una contraseña y verifique si coincide con "Secreto123".
8. Pide una temperatura actual; si es mayor a 30°C, muestra una alerta que diga "Clima caluroso".
9. Pregunta al usuario mediante un `prompt` si tiene hambre (respondiendo "sí" o "no") y muestra un consejo diferente para cada caso.
10. Solicita un número y determina si es múltiplo de 5.
11. Crea un sistema de notas: 90+ es "A", 80-89 es "B", 70-79 es "C" y menos de 70 es "F" usando `else if`.
12. Implementa un menú de opciones (1. Café, 2. Té, 3. Chocolate) y usa `switch` para mostrar el precio de cada uno.
13. Pide un número del 1 al 7 y muestra el nombre del día de la semana correspondiente (1 = Lunes, etc.).
14. Determina si un número ingresado es positivo, negativo o exactamente cero usando `else if`.
15. Solicita el nombre de un mes y responde a qué estación del año pertenece (Primavera, Verano, Otoño, Invierno).
16. **Anidación:** Pide al usuario su edad y si tiene licencia de conducir (si/no). Solo si tiene ambas condiciones, di que puede conducir.
17. Crea una calculadora básica que pida dos números y una operación (+, -, *, /) y devuelva el resultado usando `switch`.
18. Pide una hora (formato 0-23) y muestra un saludo: "Buenos días" (6-12), "Buenas tardes" (13-18) o "Buenas noches".
19. Escribe un algoritmo que determine si un año ingresado por el usuario es bisiesto.
20. Pide tres medidas de lados y determina si el triángulo es equilátero, isósceles o escaleno.
21. Define un `array` con 5 nombres de frutas y muéstralo completo en la consola.
22. Crea un array de colores y muestra en un `alert` únicamente el tercer elemento (índice 2).
23. Declara un array de estudiantes y muestra en consola cuántos elementos tiene usando la propiedad `.length`.
24. Crea un array de números del 1 al 5 y cambia el valor del primer elemento a 0.
25. Agrega un nuevo elemento al final de un array de deportes utilizando un método de array o asignación por índice.
26. Crea un array vacío y utiliza tres `prompt` consecutivos para llenarlo con nombres de países.
27. Muestra el último elemento de un array de ciudades utilizando la propiedad `.length - 1`.
28. Declara un array con tres números y muestra la suma del primer y el segundo elemento.
29. Crea un array de marcas de laptops y verifica si la lista tiene más de 3 elementos para mostrar un aviso.
30. Une todos los elementos de un array de animales en un solo mensaje de texto y muéstralo con `alert`.
31. Usa el método `.includes()` para verificar si el color "Verde" existe en un array de colores predefinido.
32. Pide un nombre al usuario y verifica si ese nombre se encuentra en un array de "Invitados VIP".
33. Crea un array de números y verifica si el número 10 está presente; muestra un mensaje de éxito si lo encuentra.
34. Si la longitud (`.length`) de un array de tareas pendientes es mayor a 5, muestra "Tu agenda está llena".
35. Verifica si un usuario tiene el rol de "Editor" dentro de un array llamado `permisos`.
36. Pide una palabra y comprueba si existe dentro de un array que actúe como diccionario simple.
37. Si un array de compras incluye "Leche", agrega automáticamente "Cereal" al mismo array.
38. Declara un array de materias escolares y usa `.includes()` para saber si "Programación" es parte de la lista.
39. Crea un array con números de lotería ganadores y verifica si el número del usuario es uno de ellos.
40. Verifica si un array de frutas **no** incluye la palabra "Pera" y, de ser así, agrégala.
41. **Login complejo:** Pide usuario y clave. Si el usuario es "admin", pide la clave; si la clave es "123", da acceso.
42. Pregunta una fruta; si está en tu lista muéstrala, de lo contrario di "No disponible".
43. Usa `switch` para evaluar una letra (A, B, C, D) y asignarle un significado de desempeño escolar.
44. Pide al usuario un número de índice (0-4) y muestra el valor que ocupa esa posición en un array de 5 elementos.
45. Comprueba si la cantidad de elementos en un array es un número par o impar.
46. Crea una lista de palabras prohibidas; si el mensaje del usuario incluye una, muestra "Mensaje bloqueado".
47. Determina el rango de un jugador: si tiene > 1000 puntos es "Pro", si no, es "Novato" (usa operador ternario).
48. Suma dos valores de un array de precios; si el resultado es mayor a 50, aplica un "descuento" virtual.
49. Usa `switch` para convertir tallas: "S" = Pequeña, "M" = Mediana, "L" = Grande.
50. Pide al usuario una posición y un nuevo valor, y reemplaza el dato en un array existente.
51. Validador: Verifica que un nombre ingresado no esté vacío y que tenga al menos 3 caracteres usando `.length`.
52. **Cajero:** Pide un PIN; si es correcto, muestra un menú con "Retirar" o "Depositar" usando `switch`.
53. Antes de agregar un elemento a una lista, usa `.includes()` para asegurar que no esté repetido.
54. Crea un conversor de divisas: el usuario elige la moneda (Dólar, Euro) y se hace el cálculo con `if`.
55. Trivia: Haz una pregunta al usuario; si responde correctamente, suma un punto a una variable.
56. Inventario: Si un array de productos tiene menos de 2 elementos, muestra "Stock crítico".
57. Filtro: Si un array de comentarios incluye la palabra "spam", muestra una advertencia.
58. Cine: El precio de la entrada es 10; si el usuario tiene un cupón (verificar en array), el precio es 7.
59. Verifica si el nombre de un usuario termina con la letra "a" (pista: usa `.length - 1` para ver la última letra).
60. Semáforo: Pide un color (rojo, amarillo, verde) y usa `switch` para decir la acción a tomar.
61. Estudiante: Si el promedio es > 70 y la asistencia es > 80%, está aprobado (usa anidación).
62. En un array de usuarios registrados, verifica si el primer elemento (índice 0) es el administrador.
63. Préstamo: Pide salario y edad. Si es mayor de 21 y gana más de 2000, muestra "Aprobado".
64. Busca si el número 7 está en un array y muestra un mensaje confirmando su hallazgo.
65. Puerta digital: Requiere que un nombre esté en la lista de "autorizados" y que el código sea correcto.
66. Usa `switch` para categorizar animales: "Perro" -> Mamífero, "Loro" -> Ave, "Serpiente" -> Reptil.
67. Clasifica una palabra como "Corta" si tiene menos de 5 letras o "Larga" si tiene 5 o más.
68. Recomendación: Si el usuario elige "Acción", recomienda una película de un array de acción.
69. Verifica si una lista de tareas está vacía comparando su `.length` con 0.
70. **Proyecto Final:** Crea un flujo donde pidas nombre, valides si está en un array de usuarios y le des la bienvenida con su nombre.
