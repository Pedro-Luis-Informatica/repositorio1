# 💻 Curso de Desarrollo Web

Bienvenido al repositorio oficial del **Curso de Desarrollo Web**.  
En este espacio encontrarás **materiales, actividades, prácticas y exámenes** para desarrollar tus habilidades en programación web.

---

## 📚 Descripción del Curso

Este curso está diseñado para que los estudiantes aprendan desde **los fundamentos de la informática aplicada al desarrollo web** hasta la creación de **páginas web dinámicas y modernas**.

Durante el curso aprenderás a utilizar tecnologías fundamentales del desarrollo web:

- HTML
- CSS
- JavaScript
- Git y GitHub
- Principios de diseño web
- Buenas prácticas de programación

---

## 🎯 Objetivos del Curso

Al finalizar el curso el estudiante será capaz de:

- Comprender cómo funciona la web.
- Crear páginas web utilizando **HTML y CSS**.
- Programar funcionalidades interactivas con **JavaScript**.
- Utilizar **Git y GitHub** para control de versiones.
- Desarrollar proyectos web básicos.
